# Resume Web App

This is a Node.js web application for displaying a resume using Express and Handlebars.

## Installation

1. **Clone the repository** (if you haven't already):
	```bash
	git clone https://github.com/heng-vicheka/hw6-resume
	cd hw6-resume
	```

2. **Install dependencies:**
	```bash
	npm install
	```

## Running the Application

Start the server with:

```bash
npm start
```

The application will run on [http://localhost:3000](http://localhost:3000) by default.

## Project Structure

- `app.js` - Main application file
- `data.json` - Resume data
- `views/` - Handlebars templates
- `public/` - Static files (CSS)

## Dependencies

- express
- express-handlebars

Install all dependencies with `npm install` as shown above.

---
Feel free to modify the resume data in `data.json` and the templates in `views/` to customize your resume.
